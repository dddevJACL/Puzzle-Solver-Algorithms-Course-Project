# Puzzle-Solver-Algorithms-Course-Project
Part of project 8 for Analysis of Algorithms course (Fall 2023)
Uses BFS to find the shortest path in a 2-d array puzzle board such as this one
```python
puzzle = [
['-', '-', '-', '-', '-'],
['-', '-', '#', '-', '-'],
['-', '-', '-', '-', '-'],
['#', '-', '#', '#', '-'],
['-', '#', '-', '-', '-']
]
```
